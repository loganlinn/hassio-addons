## Installation

[![Open your Home Assistant instance and show the add app repository dialog with a specific repository URL pre-filled.](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2Floganlinn%2Fhassio-addons%2F)

Add this repository to Home Assistant:

```text
https://github.com/loganlinn/hassio-addons
```

Then install the add-on from the Home Assistant add-on store.
